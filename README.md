# 🎬 Film Finder  

Film Finder is a web application that helps users **search, explore, and discover movies**. It fetches real-time movie details from the **TMDB API** and uses **Appwrite** to track frequently searched movies, generating a **Trending Movies** list based on user activity.  

🔗 **Live Demo:** [Film Finder on Vercel](https://film-finder-snowy.vercel.app/)  
📦 **Repository:** [GitHub Repo](https://github.com/ImAjay01/Film-Finder)  

---

## 🚀 Features  

- 🔍 **Movie Search**: Search any movie using the TMDB API.  
- ⭐ **Trending Movies**: Tracks frequently searched titles and shows trending results using Appwrite.  
- ⚡ **Fast & Responsive**: Built with React + Vite for speed and smooth experience.  
- ☁️ **Cloud Backend**: Appwrite used for database storage and analytics.  
- 📱 **Responsive UI**: Works across devices (desktop, tablet, mobile).  
- 🌐 **Deployed on Vercel** for seamless access.  

---

## 🛠️ Tech Stack  

- **Frontend:** React.js, Vite, TypeScript, JavaScript, CSS  
- **Backend/Database:** Appwrite  
- **API Integration:** TMDB API  
- **Deployment:** Vercel  
- **Version Control:** Git & GitHub

⚙️ Installation & Setup

Clone the repository

git clone https://github.com/ImAjay01/Film-Finder.git
cd Film-Finder


Install dependencies

npm install


Create a .env file in the root directory and add your API keys:

VITE_TMDB_API_KEY=your_tmdb_api_key
VITE_APPWRITE_ENDPOINT=your_appwrite_endpoint
VITE_APPWRITE_PROJECT_ID=your_appwrite_project_id


Run the development server

npm run dev


Open http://localhost:5173
 to view it in your browser.
